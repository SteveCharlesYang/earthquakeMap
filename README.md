# Earthquake Map

用3D地球展示地震发生点

## 想要实现的功能

* 爬取数据
* 数据展示到3D地球上
* 部署服务器，实时抓起数据，更新3D地球
* 分析数据，对数据进行可视化

## node爬取数据    

1. HTTP模块
2. cheerio模块: 服务器处理html,操作和JQuery一样
3. fs模块: 读写文件

* 处理数据，写入文件
